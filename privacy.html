FlowSync 隐私政策

最近更新：2026 年 7 月 14 日

FlowSync（以下简称"我们"）非常重视您的隐私。本政策说明我们在您使用 FlowSync 应用（以下简称"本应用"）时收集、使用和保护您信息的方式。

一、我们收集的信息

  • 账号信息：您注册时提交的用户名和密码。密码使用 bcrypt 算法加密后存储，我们无法还原明文。
  • 项目数据：您创建的项目、模块、任务内容。这些数据存储在我们的服务器数据库中。
  • API Key：您在设置页填写的第三方服务 API Key（高德地图、智谱 AI、和风天气等）仅保存在您设备的浏览器 localStorage 中，不会上传到我们的服务器。
  • 使用日志：我们可能记录访问日志（IP、时间、请求路径）用于安全防护和故障排查。

二、我们如何使用信息

  • 提供 AI 任务拆解、项目管理等核心功能。
  • 在您授权调用第三方服务时（如查询地图、天气、航班），将必要参数转发给对应服务。
  • 改进产品体验，修复缺陷。
  • 防范滥用、保障服务安全。

我们不会将您的个人信息出售或用于商业广告推送。

三、信息存储位置

数据类型存储位置是否上传服务器
用户名 / 密码服务器数据库是（密码已加密）
项目 / 模块 / 任务服务器数据库是
第三方 API Key设备浏览器 localStorage否
登录凭证（Token）设备浏览器 localStorage否

四、第三方 SDK / 服务

本应用在您主动使用相关功能时，会调用以下第三方服务。这些服务可能依据其自身隐私政策处理必要数据：

服务用途传输数据隐私政策
高德地图 地点搜索、路线规划 您输入的地点关键字 查看
智谱 AI AI 任务拆解生成 您输入的项目描述文本 查看
和风天气 天气查询 您输入的城市名称 查看

上述第三方调用使用的 API Key 由您自行配置并保存在本地，我们不会代为保管或转发您的 Key。

五、信息安全

  • 密码使用 bcrypt 加盐哈希存储。
  • 登录态使用 JWT，密钥仅存于服务器环境变量。
  • API 接口启用速率限制，防范暴力破解。
  • 生产环境强制 HTTPS 传输。

尽管我们采取合理措施保护您的信息，但互联网传输无法保证 100% 安全。

六、您的权利

  • 访问与导出：您可随时查看自己的项目数据。
  • 删除：您可以删除项目、清空任务；如需注销账号，请联系我们。
  • 清除本地数据：在应用设置中清除浏览器数据即可删除本地 API Key 和登录凭证。

七、未成年人保护

本应用面向一般用户，不面向 14 岁以下儿童。若您是未成年人，请在监护人指导下使用并提交信息。

八、政策变更

我们可能更新本政策。重大变更时会在应用内提示。继续使用即视为接受更新后的政策。
